# rage_math

rage_math - simple math library for using in OpenGL 3+ projects. This library is part of [rage engine](https://github.com/Enziferum/rage) collection.

## requirements

- c++ 11 compiler
- cmake

## how-to
```shell
  git clone
  cd rage_math
  mkdir build && cd build
  cmake .. -G "" -DBUILD_TESTS=ON
  make 
```
